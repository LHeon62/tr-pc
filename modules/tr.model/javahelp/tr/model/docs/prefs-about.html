User Preferences

The Preferences window enables you to change any of the application's configurable settings. The Preferences window offers two views for changing settings.

Basic preferences view (default). In the Basic view, you can select either the General, Miscellaenous or Actions Screens category. These categories are explained in the following sections. When you change settings in Basic preferences, the changes are not applied until you click OK.

The Advanced Options are not normally needed and should only by used by advanced users or in exceptional cases.
