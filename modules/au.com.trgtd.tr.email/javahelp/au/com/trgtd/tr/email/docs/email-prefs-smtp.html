Email SMTP

A SMTP (Simple Mail Transfer Protocol) server account is needed for sending email. This is required for the team delegation and response functionality. You should use an account from a provider such as Gmail. Your provider should publish the appropriate settings to use.

Set the SMTP account details:
SMTP Server
The host name of the outgoing SMTP server, such as smtp.gmail.com.
Username
Your account username
Password
Your account password
Port
The port number used by the outgoing mail server. Common port numbers for outgoing mail are 25, 465, and 587.
SSL
Does the outgoing mail server support SSL (Secure Sockets Layer) or TLS (Transport Layer Security)?

Test connection to the SMTP server using the aptly named button. A check or cross icon will appear indicating success or failure.

Set the desired method for sending delegation and response email:

Email client
Always use the default system email client application
Direct send
Always directly send email (using SMTP)
Direct send to team only
Directly send email to team members only and use the email client for non-team delegates
