Android Sync Preferences

The following preferences can be changed for Android device synchronization.
  • Sync with Android devices Allows you to specify whether data synchronization with Android devices should be attempted.
  • WiFi Port Allows you to specify the TCP port used for WiFi synchronization.
    The port number must be between 49152 and 65535 (defaults to 64641).
    It should not normally need to be changed.
  • USB Port Allows you to specify the TCP port used for USB synchronization.
    The port number must be between 49152 and 65535 (defaults to 64642).
    It should not normally need to be changed. If you do change it, be sure to also change the setting in the Android application.
  • ADB Program Allows you to enter the file system path to the ADB program executable. Alternatively you can use the "Browse" button to find it.

    To synchronize using USB you need to have an executable ADB (Android Debug Bridge) program on your computer and referenced in this field.

    This field should default to the ADB program provided. However this may not be executable or have sufficient user privileges. If this is the case you will need to do one of the following:
    • set this program file as executable for your user account.
    • download and install the Android SDK and set this field to the adb (or adb.exe) file in the platform-tools folder of the installation.
